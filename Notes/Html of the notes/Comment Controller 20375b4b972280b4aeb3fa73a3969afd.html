Comment Controller

Comment on the Post : Take the content from the body , postId from the params and userid from the req.user.

create Comment with the content , post id, user id.

add the comment id to the comments array in the post and also increase the comment count.

Update Comment : Take the content from the body, check if the user id is same as the comment owner(commented By) are same. ( params will include both comment id and post id)

than update the comment and save.

Delete Comment : Find the comment and post id from params. check if user id is equal to commentedBy id and user id is equal to the post owner id.

than delete the comment and also delete the comment id from the post db and decrease the comment count.

Summery :

This page describes the comment controller functions:

  • Create Comment:
    • Takes content from body, post ID from params, and user ID from req.user
    • Creates comment with required details
    • Updates post's comment array and count
  • Update Comment:
    • Verifies comment ownership
    • Updates comment content
    • Requires both comment and post IDs
  • Delete Comment:
    • Verifies user is either comment owner or post owner
    • Removes comment
    • Updates post's comment array and count
